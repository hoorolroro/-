본인 역할 : 데이터 전처리 및 시각화

1. 전처리

(1)groupby 함수의 sum()메소드를 사용하여, 지역별로 범죄 대분류를 묶고, 범죄 중분류는 drop처리 하였다.



(2)filter 함수를 통해 지역별로 나눠진 데이터프레임을 광역시 별로 묶었다.

(3)각 지역별 범죄 비율 데이터 만들기 : 인구수에 관한 데이터를 기존 데이터에 합쳤다.(출처:통계청)
